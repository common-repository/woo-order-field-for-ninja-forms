Woocommerce Orders Dropdown field for Ninja form.

== Description ==


This plugins adds Woocommerce Orders dropdown selection field for Ninja forms for logged in users, useful for support form.

Note: This plugin is not compatible for Ninjaform 3.x, we are working to release a new version.

Our other plugins for Ninja Form plugin

* [Woo Product Field for Ninja Forms](https://wordpress.org/plugins/woo-product-field-for-ninja-forms/)

== Installation ==

This section describes how to install the plugin and get it working.

Upload ninja-forms-wooorders-field.zip file through WordPress admin, or

FTP:
1. Upload `ninja-forms-wooorders-field` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
